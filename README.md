A fully responsive hotel website designed to showcase hotel services, destinations, and contact details with a modern UI/UX.

----------------------------------------------------------------------------------------------------------------------------

Features

Fully responsive layout for desktop, tablet, and mobile

Home page with hero section

About section to describe the hotel

Destination section with multiple destinations

Single destination detail page/section

Gallery section for hotel images

Footer with contact details and links

----------------------------------------------------------------------------------------------------------------------------

Technology Stack

Frontend: HTML, CSS, JavaScript, React.js

UI/UX & Components: Bootstrap, Bootstrap Icons, Font Awesome

----------------------------------------------------------------------------------------------------------------------------

Quickstart

Clone the repository

Install dependencies

npm install


Start the development server

npm start

----------------------------------------------------------------------------------------------------------------------------

Security & Configuration

This is a frontend-only project, so no backend or sensitive credentials are used

All assets are local or from trusted CDN sources
